
# スペース
- [https://v5.getbootstrap.com/docs/5.0/utilities/spacing/](https://v5.getbootstrap.com/docs/5.0/utilities/spacing/)
- 下記1,2,3の組み合わせ

1. マージンか、パディングか
    - m : margin
    - p : padding
2. 指定する位置
    - l,r,t,b
    - x : left + right
    - y : top + bottom
3. 幅を広げるレベル
    - 0,1,2,3,4,5,auto
    - scssなどで細かく調整できる
